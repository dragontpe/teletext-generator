## NOTES

- The number after # (0, 1, 6, 8, etc.) encodes the character set.
  Lower 3 bits = SAA505x character set variant. Bit 4 = black foreground enabled.
  0 = English, 8 = English + black foreground.
- The data after the colon is base64url-encoded 7-bit character data for 25 rows.
- To view any URL, paste it into a browser. The edit.tf editor will render it.
- For clipart extraction: open the page, select the region of interest, and copy.
- Many full-page artworks contain reusable elements (borders, logos, decorative fills).
